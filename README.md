# mich·fakeCard

A lightweight **credit card test data generator** built for devs & QA.

## What It Does
- Generates random **fake card numbers** for VISA, MasterCard, Amex, Discover  
- Shows **cardholder name**, **expiry date**, **CVV**, and full formatted output  
- Saves recent cards locally (client-side)
- For **testing & development only** — not for fraud

## Tech Stack
- HTML5
- CSS3
- Vanilla JavaScript

## How to Use
1. Open `index.html` in your browser.  
2. Pick a card type or random.  
3. Click **Generate** to produce test card data.  
4. Copy or use results in your dev/testing workflows.

## Notes
- Runs 100% locally — no backend, no tracking
- Designed for dev QA and API/testing fixtures
- Not for real financial use
